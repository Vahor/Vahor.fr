    **Goals**
    - Chaque centime doit être assigné à une catégorie
    - On peut avoir des objectifs,
        - avec date de fin / début / montant
    - Les catégories fonctionnent sur le systeme d'enveloppes
    - On peut programmer des transactions (ex: loyer, salaire)
    - On peut importer des transactions (ex: csv, relevé bancaire)
    - On peut exporter des transactions (ex: csv)
    - On peut avoir des catégories de catégories (ex: loisirs > jeux vidéos)
        - max 1 niveau
    - Design en une seule page
        - Modals et shortcuts
        - Lazy loading
    
    **Features**
    - "Meteo" du mois, qui indique si on est dans le vert ou le rouge
    - AI analyse
    - Graphiques
        - Radar chart pour les catégories
        - Line chart pour les transactions (par catégorie / par mois (ou semaine, année ou total)
    - Shortcuts pour naviguer / ajouter des transactions
    - Assigner automatiquement un logo à une transaction (ex: netflix)
    - Plusieurs personnes sur un compte
    - Light / dark mode
    - Comme la personne va surement rater des transactions, on peut ajouter un bouton "set balance" qui va recalculer les transactions manquantes
        - une seule transaction avec type "balance"

    **Technologies**
    - Next.js (frontend and backend)
    - Tailwindcss (design)
    - Clerk (auth)
    - Turso (database)
    - Nivo.rocks (charts)
    - Libre Baskerville (font) - logo (just a "b")
    - Inter (font) - text (tabular-nums for numbers)
    - Stripe (paiement)
    
    Les choix de technologies sont surtout pour tester des nouvelles choses, et pour voir si je peux les utiliser dans d'autres projets.
    Par exemple, turso n'est peut être pas le meilleur choix de base de donnée (sqlite ne gere pas les floats).

    **Usage flow**
    - Créer un compte
    - Créer un objectif
    - Ajouter des transactions (entrées d'argent)
    - Assigner les transactions à un objectif
    - Ajouter des catégories
    - Ajouter des transactions (dépenses)

    **Cout final estimé**: 28€/mois
    - 20€ pour nextjs
    - 8€ pour turso
